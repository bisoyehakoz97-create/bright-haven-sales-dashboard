# Bright Haven Stores Sales Dashboard
## Project Overview 
This project analyzes the sales performance of Bright Haven Stores for May 2026 using Power BI. The dashboard was designed to help stakeholders understand revenue trends, customer behavior, regional performance, and sales team effectiveness.
# Tools Used
Power BI
Microsoft Excel
## Key Insights
- Electronics generated the highest revenue.
- Abuja and Lagos were the strongest-performing regions.
- Laptops and phones were the best-selling products.
- Customer ratings averaged 3.9 out of 5.
  ## Recommendations
- Maintain stock levels for top-selling products.
- Improve performance in low-performing regions.
- Address customer feedback to improve ratings.
